# 颜多多服务端

除敏感配置外，包含颜多多服务端全部源码及文档，仅供学习参考。

项目基于 [Egg.js](https://eggjs.org/zh-cn/) 构建。

## 快速开始

### 本地开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 部署

```bash
$ npm start
$ npm stop
```

### 内置指令

- 使用 `npm run lint` 来做代码风格检查。
- 使用 `npm test` 来执行单元测试。
- 使用 `npm run autod` 来自动检测依赖更新，详细参见 [autod](https://www.npmjs.com/package/autod) 。
